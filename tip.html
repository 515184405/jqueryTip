jqueryTip.js信息提示文档说明 此组件为PC端信息提示组件

jqueryTip.js是基于jQuery开发的一款轻量级信息提示组件

评分

如果觉得好用，就在github上 star Fock 或本站 留言 哦

浏览器兼容

浏览器兼容，下面是我们的主要兼容目标

1、IE8 或者 IE8以上 (Windows), IE8以下浏览器不兼容

2、Safari (Mac)

3、Chrome (Windows, Mac, Linux)

4、Firefox (Windows, Mac, Linux)

5、谷歌内核(webkit)浏览器，如360浏览器，搜狗浏览器，QQ浏览器等

核心方法（配置）: jqueryTip(opts)

opts是一个对象，它包含了以下key: '默认值'

javascript 代码:

						
				'bgColor' : '#000',// 背景颜色
				'color'   : '#fff', // 字体颜色
				'attribute' : 'data-tip', //要显示文字的属性名
				'target'  : 'mouseover',//事件
				'maxWidth': '',//最大的宽度
				'addClass': '', //添加类名
				'padding' : 8 , //弹出框之间的间隙
				'width'   : 'auto', //设置宽度
				'animateTime' : 300, //动画时间
				'animateType' : 'fade',//动画类型
				// 水平和垂直只能一个为center，若两个同时为center，vertical的center将被改为top
				'horizontal'  : 'center',  //水平方向  left center right
				'vertical'    : 'top',  //垂直方向  top center bottom
				'closeType'   : 'mouseout' ,   //关闭类型 auto 自动关闭 或者鼠标移除关闭
				'closeTime'   : 2500,  //关闭时间
示例 horizontal、vertical 参数演示

参数配置

				$("#tip-api-demo1").jqueryTip({
					'horizontal'  : 'center', //left、right
					'vertical'    : 'top',	//center、bottom
				})
示例 bgColor、color、attribute、target 参数演示

参数配置 注意：此处触发事件为click

				$("#tip-api-demo2").jqueryTip({
					'bgColor'  : '#f00', 
					'color'    : '#0f0',
					'attribute': 'data-title',
					'target'   : 'click',
				})
示例 maxWidth、addClass、padding、width、animateTime 参数演示

参数配置 

				$("#tip-api-demo3").jqueryTip({
					'maxWidth' : '300',
					'addClass'   : 'tipClass',
					'padding'   : '15',
					'animateTime':'700',
				})
示例 animateType、closeType、closeTime 参数演示

参数配置 

				$("#tip-api-demo4").jqueryTip({
					'animateType' : 'bounce',
					'closeType'   : 'auto',
					'closeTime'   : '5000'
				})

基本参数

bgColor - String

默认：#000

释义：设置背景颜色

color - String

默认：#fff

释义：设置文本颜色

attribute - String

默认：data-tip

释义：设置属性名

target - String

默认：mouseover

选项：mouseoverclick

释义：设置事件

maxWidth - Number/String

默认：''

释义：设置最大宽度

addClass - String

默认：''

释义：添加class

padding - Number/String

默认：8

释义：设置内边距

width - Number/String

默认：auto

释义：设置宽度

animateTime - Number/String

默认：300

释义：设置动画时间

animateType - String

默认：fade

选项：fadebounce

释义：使用动画

Bug：在IE8下动画效果失效

horizontal - String

默认：center

选项：leftcenterright

释义：设置水平方向

vertical - String

默认：top

选项：topcenterbottom

释义：设置垂直方向

closeType - String

默认：mouseout

mouseoutauto

释义：关闭类型 auto 自动关闭 或者鼠标移除关闭

closeTime - Number/String

默认：2500

释义：如果closeType == 'auto'时,此参数生效
